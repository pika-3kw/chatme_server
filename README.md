# Chatme Server

## Cài đặt và chạy server ở môi trường development

1. Cài đặt

```
$ yarn install
$ copy .env.default .env
```

2. Chỉnh sửa file .env theo hướng dẫn

3. Chạy server

```
$ yarn dev
```
